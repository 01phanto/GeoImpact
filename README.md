# GeoImpact - Geopolitical Analysis Dashboard

## 🎯 **Project Overview**
Professional geopolitical analysis platform focusing on how global events impact India across economic, political, educational, and strategic dimensions.

## 🚀 **Quick Start**

### Development Server
```bash
npm run dev
# Open http://localhost:5173
```

### Production Build
```bash
npm run build
npm run preview
```

## 🛠️ **Technology Stack**
- **React 19** - Modern component-based UI
- **Vite** - Fast build tool and dev server
- **Tailwind CSS** - Utility-first styling
- **Framer Motion** - Smooth animations
- **Recharts** - Data visualization
- **React CountUp** - Animated counters
- **Lucide React** - Professional icons

## 📊 **Features**
- **Interactive Dashboard** with live charts and metrics
- **News Analysis** with search and filtering
- **Critical Events Monitor** with detailed impact assessments
- **Study Abroad Impact** analysis with country comparisons
- **Dark Mode** support throughout the application
- **Responsive Design** optimized for all devices

## 🎨 **Component Structure**
```
src/components/
├── Dashboard.jsx          # Main metrics and charts
├── Header.jsx             # Navigation and controls
├── ImpactCards.jsx        # Critical events analysis
├── NewsSection.jsx        # News filtering and display
└── StudyAbroadSection.jsx # Education impact analysis
```
